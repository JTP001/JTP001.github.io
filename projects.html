Personal CRM Project

A group project in one of my computer science courses where we made a Personal Customer Relationship Management system.

This project was made with a real industry partner who asked for it to look this way with these functions, who is continuing to use it.

The web app was made with a REACT frontend and a Django backend with a PostgreSQL database, which were deployed using Netlify and Render.

It allows you to do a variety of things such as pull contacts, calendar events and data from other social sites and manage everything about your network in one place.

However, the project did have limitations since everything had to be free, so the connection to other social apps is limited and the functionality is there but not fully connected withouta plan with those sites to use their APIs.

Calendar App Project: OneOnOnes

A group project in one of my computer science courses where we made a website for teachers to schedule meetins with students.

The website was made with a REACT frontend and a Django backend to manage an sqlite database.

The website allowed you to create schedules, add contacts and invite them to those schedules, make suggested schedules based on a greedy algorithm, and more.
