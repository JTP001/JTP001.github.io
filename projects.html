Calendar App Project: OneOnOnes

A group project in one of my computer science courses where we made a website for teachers to schedule meetins with students.

The website was made with a REACT frontend and a Django backend to manage an sqlite database.

The website allowed you to create schedules, add contacts and invite them to those schedules, make suggested schedules based on a greedy algorithm, and more.
